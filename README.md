# SkillBridge

SkillBridge is a micro-learning platform designed to help users build skills through bite-sized lessons and quizzes.

## Features
- User authentication (login and registration)
- Explore and track micro-learning topics
- Create and share learning content
- Progress tracking

## Technologies
- Backend: Node.js, Express, MongoDB
- Frontend: React

## Installation
1. Clone the repository: `git clone <repository_url>`
2. Navigate to the backend folder and run: `npm install`
3. Navigate to the frontend folder and run: `npm install`

## Usage
1. Start the backend server.
2. Start the frontend development server.
